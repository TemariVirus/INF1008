Suppose that Q is a Queue. List the content of the queue after each operation and
show the output value if a value is returned from the operation.

| Operation     | Output | [Front – Queue – Rear] |
| ------------- | ------ | ---------------------- |
| Q = Queue()   | None   | []                     |
| Q.isEmpty()   | True   | []                     |
| Q.enqueue(8)  | None   | [8]                    |
| Q.enqueue(-5) | None   | [8, -5]                |
| Q.dequeue()   | 8      | [-5]                   |
| Q.enqueue(2)  | None   | [-5, 2]                |
| Q.peek()      | -5     | [-5, 2]                |
| Q.dequeue()   | -5     | [2]                    |
| Q.isEmpty()   | False  | [2]                    |
| Q.peek()      | 2      | [2]                    |
| Q.dequeue()   | 2      | []                     |
